Planty ## Description Planty est un site de boisson énergisante sur ce site on peux passer commande, prendre contact avec l'équipe et découvrir les différents gouts de cette boisson ## Table des Matières - [Fonctionnalités](#fonctionnalités) - [Technologies Utilisées](#technologies-utilisées) - [Structure du Projet](#structure-du-projet) ## Fonctionnalités - **Navigation Intuitive** : Menu de navigation clair pour accéder facilement aux différentes sections du site. - **Page d'Accueil** : Présentation des services ou produits proposés. - **Nous Rencontrer** : Informations sur l'équipe et coordonnées pour un contact facile. - **Commander** : Page avec des formulaires pour passer des commandes en ligne. - Formulaire de contact pour les demandes d'informations. - Deux formulaires dédiés pour passer des commandes. - **Accès Administrateur** : Lien sécurisé pour les administrateurs connectés. - **Mentions Légales** : Lien vers une page prête à accueillir les mentions légales. - **Design Réactif** : Adapté à tous les types d'écrans pour une expérience utilisateur optimale. ## Technologies Utilisées - HTML, CSS, PHP **Système de Gestion de Contenu** : WordPress - **Thème Enfant** : Utilisation d'un thème enfant pour personnaliser et étendre les fonctionnalités du thème principal. - **Hooks** : Implémentation de hooks pour modifier ou ajouter des fonctionnalités de manière efficace. ## Structure du Projet Le site est organisé en plusieurs pages principales, chacune ayant un rôle spécifique : - **Home (Accueil)** : Page d'accueil du site qui présente une vue d'ensemble des produits proposés. - **Nous Rencontrer** : Page contenant des informations sur l'équipe de l'entreprise, avec un formulaire de contact - **Commander** : Page dédiée aux commandes, incluant : - Deux formulaires spécifiques pour passer des commandes. ( premier formulaire pour passer commande et le deuxième pour renseigner les informations pour la livraison ) - **Lien Administrateur** :Le lien Admin du menu qui ne doit être affiché que lorsque l’internaute est connecté sur WordPress. - **Mentions Légales** : Lien dans le footer menant à une page actuellement vide, prévue pour contenir les mentions légales une fois finalisées.
